# 完整的机械臂抓取演示程序

这个程序演示了机械臂的完整抓取流程，包括自动路径规划、力反馈控制和实时状态监控。

## 功能特性

### 🤖 自动抓取序列
1. **移动到球的上方** - 机械臂自动计算逆运动学到达球的上方准备位置
2. **张开夹爪** - 夹爪张开到合适宽度准备抓取
3. **下降到球的位置** - 精确下降到球的接触位置
4. **闭合夹爪抓取** - 基于力反馈的智能抓取，检测抓取成功
5. **抬起球** - 将球抬起到安全高度
6. **移动到新位置** - 将球移动到指定新位置
7. **放下球** - 张开夹爪释放球
8. **返回初始位置** - 机械臂返回home位置

### 📹 实时视觉反馈
- **眼在手RGB摄像头** - 实时显示末端执行器的视角
- **动态视角更新** - 摄像头跟随机械臂移动自动调整方向

### 📊 实时状态监控
- **关节角度显示** - 实时显示所有关节的角度（弧度和角度）
- **力反馈传感器** - 实时显示夹爪的接触力
  - 🔴 红色: 无接触 (< 1N)
  - 🟠 橙色: 接触检测 (1-5N)
  - 🟢 绿色: 成功抓取 (> 5N)

### 🎮 交互控制
- **自动演示模式** - 程序自动执行完整抓取序列
- **实时监控模式** - 演示完成后进入实时状态监控
- **GUI界面** - PyBullet图形界面显示3D场景

## 使用方法

```bash
cd Synria-Robot-Descriptions-main
python examples/complete_grasping_demo.py
```

## 技术实现

### 运动规划
- **逆运动学计算** - 使用PyBullet内置的逆运动学求解器
- **平滑轨迹** - 余弦插值实现平滑关节运动
- **碰撞检测** - 自动避开障碍物

### 力控制
- **力传感器集成** - 启用关节力矩传感器
- **智能抓取** - 基于力阈值的抓取成功检测
- **实时力反馈** - 连续监控抓取力

### 视觉系统
- **眼在手配置** - 摄像头安装在末端执行器上
- **自动对焦** - 摄像头自动朝向运动方向
- **实时渲染** - 硬件加速的RGB图像渲染

## 故障排除

### 常见问题
1. **逆运动学失败** - 检查目标位置是否在工作空间内
2. **抓取失败** - 调整力阈值或球的位置
3. **摄像头无图像** - 检查OpenGL驱动支持

### 调试信息

- 关节角度变化
- 力传感器读数
- 运动规划状态
- 错误信息
